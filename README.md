# LazyWorker
LazyWorker for lazy developers 

## For using it locally


Front - React
Back - NestJS
Database - Postqres


### Client 

1) npm install
2) npm run start


### Server

1) npm install
2) npm run start


### Database

1) psql -U (username)
2) (password)
3) CREATE DATABASE (db_name);
